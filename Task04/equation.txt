Чек-лист тестирования функции нахождения корней уравнения ax^2 + bx + c = 0

1. Два различных действительных корня (D > 0)
Входные данные: a = 1, b = -3, c = 2
Ожидаемый результат: [1.0, 2.0]
Описание: Стандартный случай квадратного уравнения с двумя корнями. Проверка их сортировки по возрастанию.

2. Один действительный корень (D = 0)
Входные данные: a = 1, b = -2, c = 1
Ожидаемый результат: [1.0]
Описание: Дискриминант равен нулю, корень один.

3. Нет действительных корней (D < 0)
Входные данные: a = 1, b = 1, c = 1
Ожидаемый результат: [] (пустой список)
Описание: Уравнение не имеет решений в области действительных чисел.

4. Линейное уравнение (a = 0, b != 0)
Входные данные: a = 0, b = 2, c = -4
Ожидаемый результат: [2.0]
Описание: Вырождение в линейное уравнение bx + c = 0.

5. Константа без корней (a = 0, b = 0, c != 0)
Входные данные: a = 0, b = 0, c = 5
Ожидаемый результат: []
Описание: Неверное равенство (5 = 0), корней нет.

6. Бесконечное множество решений (a = 0, b = 0, c = 0)
Входные данные: a = 0, b = 0, c = 0
Ожидаемый результат: None (или специфичное исключение/флаг)
Описание: Тождество 0 = 0, решением является любое число.

7. Неполное квадратное уравнение (a ≠ 0, b = 0, c ≠ 0)
Входные данные: a = 1, b = 0, c = -4
Ожидаемый результат: [-2.0, 2.0]
Описание: Уравнение вида ax² + c = 0 с двумя корнями.

8. Неполное квадратное уравнение (a ≠ 0, b = 0, c = 0)
Входные данные: a = 2, b = 0, c = 0
Ожидаемый результат: [0.0]
Описание: Уравнение вида ax^2 = 0 с одним корнем x = 0.

9. Отрицательный старший коэффициент (a < 0, D > 0)
Входные данные: a = -1, b = 0, c = 4
Ожидаемый результат: [-2.0, 2.0]
Описание: Сортировка корней должна выполняться корректно при отрицательном a.

10. Дробные коэффициенты
Входные данные: a = 0.5, b = -1.5, c = 1
Ожидаемый результат: [1.0, 2.0]
Описание: Проверка работы функции с числами с плавающей точкой (float).

11. Граничное значение дискриминанта (D близко к 0)
Входные данные: a = 1, b = -2, c = 0.9999
Ожидаемый результат: [0.99, 1.01] (приблизительные значения, два близких корня)
Описание: Проверка точности вычислений при дискриминанте, близком к нулю.
